test_context.exe
Tests the use of a ContextPage as a dropdown menu

CONTROLS:
UP: make the dropdown menu flow above the element
DOWN: make the dropdown flow below the element

TESTS:
1) SELECT OPTION
	- Clicking on this element should open a dropdown of options. Clicking again should close the dropdown and change the element's text to whatever was clicked

2) + button
	- Clicking this element should add an option with the text to the left as it's inner
