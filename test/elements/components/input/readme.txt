test_textbox.exe
Tests drawing user inputted text

CONTROLS:
SCROLL UP: rotate element clockwise
SCROLL DOWN: rotate element counter-clockwise
TOP 'SET' BUTTON: click to copy the text left of button to the center element
BOTTOM 'SET' BUTTON: click to set the max amount of characters to the number specified in the left of button

TESTS:
1) NO ARGUMENTS CASE
	- EXPECTED: raylib default font
	
2) FONTFILE LOCATION CASE
	- EXPECTED: load font from provided path
